# jeffrey-foo-biz

Personal site — AI Implementation, Digital Transformation & Governance.
Built with React + Vite.

## Local preview
```bash
npm install
npm run dev
```

## Build
```bash
npm run build   # outputs to dist/
```

## Deploy
Connected to Netlify. Push to `main` → auto-deploys.
Build command: `npm run build` | Publish dir: `dist`
